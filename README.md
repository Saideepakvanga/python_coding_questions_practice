# HackerRank Python Practice

This repository contains solutions to basic Python problems from HackerRank.  
All programs are written in Python 3.

## Problems Included

1. Say "Hello, World!" With Python  
2. Python If-Else  
3. Arithmetic Operators  
4. Python: Division  
5. Loops  
6. Write a Function  
7. Print Function  
8. Running Sum of 1D Array
9. Shuffle the Array
10. Kids With the Greatest Number of Candies
## How to Run

1. Clone the repository:
   git clone https://github.com/yourusername/hackerrank-python-practice.git

2. Open the folder:
   cd hackerrank-python-practice

3. Run any file:
   python3 filename.py

## Language Used
Python 3.x

Simple and beginner-friendly solutions for practicing Python basics.
